# Mapbox Maps SDK for Android — 概述

> 原文：[Maps SDK for Android Guides](https://docs.mapbox.com/android/maps/guides/)

## 简介

**Mapbox Maps SDK for Android** 是在 Android 应用中嵌入高度可定制地图的库。你可以：

- 选择 **Mapbox Standard**、**Mapbox Standard Satellite** 等官方样式，或在 Mapbox Studio 中设计自定义样式
- 在运行时修改样式、添加自定义数据、做点/线/面可视化
- 控制相机、手势、用户位置、地图查询等

当前文档对应 SDK 版本：**v11.24.3**。

## 系统要求

| 项目 | 要求 |
|------|------|
| Android SDK | API 21 及以上 |
| 设备 | 支持 OpenGL ES 3 |
| Kotlin | 1.6.0+（若使用 Kotlin） |
| Java | `sourceCompatibility` / `targetCompatibility` 为 Java 8+ |

## NDK 支持

### NDK 23（默认）

默认产物支持 NDK 23，**不支持** 16 KB 页大小。若应用需要 16 KB 页大小，请使用 NDK 27 产物。

### NDK 27（11.7.0 起）

在 artifact ID 后追加 `-ndk27`：

```gradle
implementation 'com.mapbox.maps:android-ndk27:11.24.3'
```

不需要 16 KB 页大小时可使用：

```gradle
implementation 'com.mapbox.maps:android:11.24.3'
```

## Java 互操作

SDK 使用 **Kotlin** 编写，与 Java 100% 互操作。纯 Java 项目也可调用公开 API。

## 与其他 Mapbox 产品配合

| 产品 | 用途 |
|------|------|
| **静态地图** | `MapSnapshotter` 生成地图快照 |
| **Mapbox Studio** | 在线设计样式，SDK 加载使用 |

## 使用条件

### 归属（Attribution）

- 使用本 SDK 的地图必须显示 **Mapbox 标识**
- 归属信息必须可见；可调整位置与背景色，但须清晰可读
- 不得篡改 Mapbox 标识或归属文字

### 遥测退出（Telemetry opt-out）

SDK 默认会发送匿名位置与使用数据。条款要求应用为用户提供 **单独退出 Mapbox Telemetry** 的方式；归属控件中通常已包含。若隐藏归属控件，必须提供替代的退出入口。
