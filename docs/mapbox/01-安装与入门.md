# 安装与入门

> 原文：[Get Started with the Maps SDK for Android](https://docs.mapbox.com/android/maps/guides/install/)

## 前置条件

- [Mapbox 账号](https://account.mapbox.com/)
- Android Studio
- Gradle

建议在 Android Studio 中用 **Empty Activity** 创建新项目（Aura 已有 `app` 模块时可跳过创建步骤）。

---

## 第一部分：配置凭证

### 步骤 1：配置 Public Token（Aura：`local.properties`）

应用必须配置 **Public Access Token**，才能将 Mapbox 资源用量关联到你的账号。

本项目通过 **Gradle 注入**，不把 token 提交到 Git：

1. 复制项目根目录的 `local.properties.example` 为 `local.properties`（若 Android Studio 已生成该文件，直接编辑即可）。
2. 增加一行（token 来自 [Tokens 页面](https://account.mapbox.com/access-tokens/)，以 `pk.` 开头）：

```properties
MAPBOX_ACCESS_TOKEN=pk.你的token
```

构建时 `app/build.gradle` 会生成字符串资源 `mapbox_access_token`。SDK 初始化地图时会自动读取，一般无需在代码里再写一遍。

未配置时构建仍可通过，但地图无法加载，Gradle 会输出警告。

**进阶**：

- 运行时设置：`MapboxOptions.accessToken = "pk...."`，须在 `MapView` inflate **之前** 调用，否则可能崩溃
- 生产环境建议配合 Token 轮换与服务端下发策略

### 步骤 2：（可选）配置定位权限

若要在地图上显示用户位置，在 `AndroidManifest.xml` 的 `<manifest>` 与 `<application>` 之间添加：

```xml
<!-- 粗略位置 -->
<uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" />
<!-- 仅当需要精确定位时添加；须同时有 COARSE -->
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
```

可用 SDK 的 `PermissionsManager` 检查并申请权限。详见 [14-权限处理](./14-权限处理.md)。

## 第二部分：添加依赖

### 步骤 1：添加 Mapbox Maven 仓库

编辑 **`settings.gradle`**（或 `settings.gradle.kts`），在 `dependencyResolutionManagement.repositories` 内添加（**不要**写在 `pluginManagement` 里）：

```groovy
dependencyResolutionManagement {
    repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
    repositories {
        google()
        mavenCentral()
        maven {
            url = uri("https://api.mapbox.com/downloads/v2/releases/maven")
        }
    }
}
```

### 步骤 2：添加 Maps SDK 依赖

在 **`app/build.gradle`** 的 `dependencies` 中：

```groovy
dependencies {
    // 需要 16 KB 页大小支持时使用 ndk27 变体
    implementation 'com.mapbox.maps:android-ndk27:11.24.3'
    // 否则使用默认：
    // implementation 'com.mapbox.maps:android:11.24.3'
}
```

**Jetpack Compose** 额外配置：

```groovy
android {
    buildFeatures {
        compose = true
    }
}
dependencies {
    implementation 'com.mapbox.extension:maps-compose-ndk27:11.24.3'
    // 或 maps-compose:11.24.3
}
```

执行 **Sync Project with Gradle Files**。

### 关于 Google Play Services

自 **v11.8.0** 起，SDK 会传递依赖 Google Play Services（用于 HTTP/3 客户端）。可按官方 [Removing Google Play](https://docs.mapbox.com/android/maps/guides/removing-google-play/) 指南移除（将失去 HTTP/3 支持）。

### 常见问题

- Maven 仓库写错位置 → 依赖解析失败
- Token 仍为占位符 → 地图无法加载
- 传递依赖冲突 → 使用 `exclude group` 排除

## 第三部分：添加地图

### Jetpack Compose

```kotlin
package com.catclaw.aura  // 替换为你的包名

import android.os.Bundle
import androidx.activity.ComponentActivity
import androidx.activity.compose.setContent
import androidx.compose.foundation.layout.fillMaxSize
import androidx.compose.foundation.layout.padding
import androidx.compose.ui.Modifier
import androidx.compose.ui.unit.dp
import com.mapbox.geojson.Point
import com.mapbox.maps.extension.compose.MapboxMap
import com.mapbox.maps.extension.compose.animation.viewport.rememberMapViewportState
import com.mapbox.maps.plugin.attribution.Attribution
import com.mapbox.maps.plugin.logo.Logo
import com.mapbox.maps.plugin.scalebar.ScaleBar

class MainActivity : ComponentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContent {
            MapboxMap(
                Modifier.fillMaxSize(),
                mapViewportState = rememberMapViewportState {
                    setCameraOptions {
                        zoom(2.0)
                        center(Point.fromLngLat(-98.0, 39.5))
                        pitch(0.0)
                        bearing(0.0)
                    }
                },
                scaleBar = { ScaleBar(Modifier.padding(top = 60.dp)) },
                logo = { Logo(Modifier.padding(bottom = 40.dp)) },
                attribution = { Attribution(Modifier.padding(bottom = 40.dp)) }
            )
        }
    }
}
```

### Android View（适合当前 Aura 工程）

```kotlin
package com.catclaw.aura

import android.os.Bundle
import androidx.appcompat.app.AppCompatActivity
import com.mapbox.geojson.Point
import com.mapbox.maps.CameraOptions
import com.mapbox.maps.MapInitOptions
import com.mapbox.maps.MapView
import com.mapbox.maps.plugin.attribution.attribution
import com.mapbox.maps.plugin.logo.logo
import com.mapbox.maps.plugin.scalebar.scalebar

class MainActivity : AppCompatActivity() {
    private lateinit var mapView: MapView

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        mapView = MapView(
            this,
            MapInitOptions(
                context = this,
                cameraOptions = CameraOptions.Builder()
                    .center(Point.fromLngLat(-98.0, 39.5))
                    .pitch(0.0)
                    .zoom(2.0)
                    .bearing(0.0)
                    .build()
            )
        )
        mapView.scalebar.marginTop = 200f
        mapView.logo.marginBottom = 140f
        mapView.attribution.marginBottom = 140f
        setContentView(mapView)
    }
}
```

运行模拟器/真机后应看到可缩放、平移的地球视图。
