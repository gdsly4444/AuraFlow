# 地图样式

> 原文：[Map Styles](https://docs.mapbox.com/android/maps/guides/styles/)

## 什么是地图样式

地图样式决定地图的**外观**：颜色、背景、道路、标签、POI 等。加载样式后，你可以在运行时：

- 修改样式配置
- 增删图层、改图层属性
- 用 Mapbox 表达式做数据驱动样式

## 默认样式

SDK 默认加载 **Mapbox Standard**——通用、可配置（颜色、要素可见性等）。也可：

- 在 Mapbox Studio 设计自定义样式
- 通过 **source + layer** 或 **style fragment** 导入自有数据

## Mapbox Standard 注意点

对 **Standard / Standard Satellite**，你只能直接操作**自己添加**的图层；底图本身通过 **style import 配置**（如 `setStyleImportConfigProperty`）调整，而不是随意改底图里每一层。

## 与 Mapbox Studio 的分工

- **运行时改样式**：本仓库 `docs/mapbox` 各指南
- **在 Studio 里画样式**：见 [Mapbox Studio 手册](https://docs.mapbox.com/studio-manual/)

## 样式通用概念（跨平台）

样式 = JSON 文档，包含：

- **sources**：地理数据来源（GeoJSON、矢量瓦片等）
- **layers**：如何绘制 source 中的要素
- **sprite / glyphs**：图标与字体

Android 中通过 `Style`、`MapboxMap.loadStyle`、Style DSL 操作。
