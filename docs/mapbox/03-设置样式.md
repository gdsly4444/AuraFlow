# 设置样式

> 原文：[Set a style](https://docs.mapbox.com/android/maps/guides/styles/set-a-style/)

样式决定地图视觉，并告诉 SDK 从哪拉取瓦片与数据。可在初始化时指定，也可运行时切换。

---

## 加载样式

### Mapbox Standard（默认）

不指定 `styleUri` 即使用 Standard。

**Android View：**

```kotlin
mapView = MapView(context)  // 默认 Standard
```

**显式指定：**

```kotlin
mapView = MapView(context, MapInitOptions(
    context = context,
    styleUri = Style.STANDARD
))
```

**Jetpack Compose：**

```kotlin
MapboxMap(style = { MapboxStandardStyle() })
```

### 自定义样式（Studio URL）

格式：`mapbox://styles/{username}/{style-id}`

```kotlin
mapView = MapView(context, MapInitOptions(
    context = context,
    styleUri = "mapbox://styles/your-username/your-style-id"
))
```

### 使用样式常量

| 样式名 | 常量 |
|--------|------|
| Mapbox Standard | `Style.STANDARD` |
| Mapbox Standard Satellite | `Style.STANDARD_SATELLITE` |

其他历史样式（如 `Style.OUTDOORS`）以当前 SDK 文档为准。

### 延迟加载样式

**View：** `styleUri = null`  
**Compose：** `MapStyle(style = "")`  

地图会先显示黑色背景。可用最小 JSON 只含 `background` 层改背景色：

```kotlin
private const val BLANK_STYLE = """
{
  "layers": [{
    "id": "background",
    "type": "background",
    "paint": {"background-color": "hsl(100, 50%, 50%)"}
  }]
}
"""
```

之后调用 `mapboxMap.loadStyle(...)` 加载正式样式。

### XML 中指定样式

```xml
<com.mapbox.maps.MapView
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    mapbox:mapbox_styleUri="mapbox://styles/your-username/your-style-id" />
```

## 配置样式

### Mapbox Standard 运行时配置

通过 `setStyleImportConfigProperty`，import id 为 **`basemap`**。

示例：灯光预设改为 dusk，隐藏 POI 标签：

```kotlin
mapView.mapboxMap.getStyle { style ->
    style.setStyleImportConfigProperty("basemap", "lightPreset", Value.valueOf("dusk"))
    style.setStyleImportConfigProperty("basemap", "showPointOfInterestLabels", Value.valueOf(false))
}
```

**Compose** 可在 `MapboxStandardStyle` 的 `rememberStandardStyleState` 里配置 `lightPreset`、`showPointOfInterestLabels` 等。

也可在 Mapbox Studio 里创建「导入 Standard」的自定义样式并发布 URL。

### 自定义样式图层属性

```kotlin
mapView.mapboxMap.getStyle { style ->
    style.setStyleLayerProperty("water", "fill-color", Value.valueOf("#45d9ca"))
}
```

需知道图层 id 与属性名，参见 [Mapbox Style Specification](https://docs.mapbox.com/style-spec/)。

---

## 运行时切换样式

```kotlin
mapView.mapboxMap.loadStyle("mapbox://styles/mapbox/standard-satellite")
```

**注意**：切换后，初始化后动态添加的图层需重新添加，或使用 `addPersistentLayer` 保持跨样式存在。

**Compose**：更换整个 `MapboxStyleComposable`（如 `MapboxStandardStyle` ↔ `MapStyle(url)`）。放在 `slot` / `layerPosition` 中的层会随新样式加载；`MapboxMap` content 中的层可作为 persistent layer。
