# 源（Source）与图层（Layer）

> 原文：[Work with sources and layers](https://docs.mapbox.com/android/maps/guides/styles/work-with-layers/)（节选译文）

## 核心概念

| 概念 | 作用 |
|------|------|
| **Source** | 地理数据：点线面的形状与位置 |
| **Layer** | 样式：数据如何画在地图上（颜色、线宽、图标等） |

添加图层前**必须先添加**其依赖的 source（多数图层类型要求 `sourceId`）。

入口类：`Style`（`mapView.mapboxMap.getStyle { }` / `loadStyle`）。

## 渲染顺序

- 一般：后声明的图层在上层
- **Mapbox Standard**：主要靠 **slot**（`bottom` / `middle` / `top`）决定插入位置
- 启用 **Globe** 或 **Terrain** 时，部分图层可能被批处理重排；`fill`、`line` 等会画在 symbol 之下

若 `addLayerAbove` 效果不符合预期，可尝试关闭 terrain 并使用 Mercator 投影。

## Style DSL（推荐写法）

DSL 函数名与类名对应（首字母小写），例如 `circleLayer`、`geoJsonSource`：

```kotlin
mapView.mapboxMap.loadStyle(
    style(style = Style.TRAFFIC_DAY) {
        +geoJsonSource(id = "earthquakes") {
            data(GEOJSON_URL)
            cluster(false)
        }
        +circleLayer(layerId = "earthquakeCircle", sourceId = "earthquakes") {
            circleRadius(get { literal("mag") })
            circleColor(Color.RED)
            circleOpacity(0.3)
        }
    }
) { /* 样式加载完成 */ }
```

## 运行时添加图层

```kotlin
mapView.mapboxMap.getStyle { style ->
    style.addSource(geoJsonSource("SOURCE_ID") {
        data("asset://route.geojson")
    })
    style.addLayer(lineLayer("LAYER_ID", "SOURCE_ID") {
        lineColor(ContextCompat.getColor(context, R.color.black))
        lineWidth(3.0)
    })
}
```

## 运行时更新图层

```kotlin
style.getLayerAs<FillLayer>("LAYER_ID")?.fillOpacity(0.7)
```

Compose 中：运行时添加的层用 **mutable state** 驱动属性；样式 JSON 里已有的层在 `MapEffect` 里改。

## Mapbox Standard 的 Slot

| Slot | 位置 |
|------|------|
| `bottom` | 面要素（陆地、水域等）之上 |
| `middle` | 线要素之上、3D 建筑之下 |
| `top` | POI 之上、地名/transit 标签之下；适合 symbol |
| 未指定 | 所有已有图层之上 |

```kotlin
+lineLayer(layerId = "line-layer", sourceId = "line-layer") {
    lineColor(rgb(255.0, 165.0, 0.0))
    slot("middle")
}
```

## 图层顺序（非 Standard）

```kotlin
style.moveStyleLayer("population", LayerPosition(null, "state-labels", null))
style.addLayerBelow(lineLayer(...), below = "road-label")
```

## 删除图层

```kotlin
if (style.styleLayerExists(LAYER_ID)) {
    style.removeStyleLayer(LAYER_ID)
}
```

Compose：条件渲染或 `MapEffect` 中删除。

## 常见 Source 类型

### Vector（矢量瓦片）

```kotlin
+vectorSource("SOURCE_ID") {
    url("mapbox://mapbox.mapbox-terrain-v2")
}
```

Mapbox 托管瓦片集 URL：`mapbox://{username}.{tilesetname}`。  
使用 vector source 的 layer 必须设置 **`source-layer`**。

### GeoJSON

```kotlin
+geoJsonSource("SOURCE_ID") {
    data("https://example.com/data.geojson")
    // 或 data(geoJsonString) / data("asset://file.geojson")
}
```

适合中小数据、可动态更新；大数据量性能不如 vector tiles。

### 其他

Raster、Image、Dem 等见 [Style Spec - Sources](https://docs.mapbox.com/style-spec/sources/)。

## 常见 Layer 类型

| 类型 | 用途 |
|------|------|
| `circle` | 点 → 圆 |
| `line` | 线 |
| `fill` | 面填充 |
| `symbol` | 图标、文字 |
| `heatmap` | 热力图 |
| `fill-extrusion` | 3D 挤出建筑等 |
| `raster` | 栅格 |

矢量/geojson 最常用 circle、line、fill、symbol。完整列表见 Style Spec。
