# 相机位置

> 原文：[Camera position](https://docs.mapbox.com/android/maps/guides/camera-and-animation/camera/)

**相机**即用户俯视地图的视角。SDK 通过以下属性描述相机：

| 属性 | 含义 |
|------|------|
| `center` | 中心点经纬度 |
| `bearing` | 旋转角（0° 北朝上，90° 东朝上） |
| `pitch` | 俯仰（0° 垂直俯视，越大越「看向地平线」） |
| `zoom` | 缩放（0 看大陆，~11 城市，更大看建筑） |
| `padding` | 四边内边距，影响 center 在屏幕上的位置 |
| `anchor` | 缩放/旋转锚点像素坐标；与 `center` 互斥 |

可用 [Location Helper](https://labs.mapbox.com/location-helper/) 调试参数。

---

## 设置相机

### 初始化时（XML）

```xml
<com.mapbox.maps.MapView
    mapbox:mapbox_cameraTargetLat="40.7128"
    mapbox:mapbox_cameraTargetLng="-74.0060"
    mapbox:mapbox_cameraZoom="9.0" />
```

### 初始化时（代码）

```kotlin
val camera = CameraOptions.Builder()
    .center(Point.fromLngLat(-74.0066, 40.7135))
    .pitch(45.0)
    .zoom(15.5)
    .bearing(-17.6)
    .build()

mapView = MapView(this, MapInitOptions(context = this, cameraOptions = camera))
```

未指定时：若样式 JSON 有相机字段则用样式；否则 `0,0`、zoom 0。

### 地图加载后

```kotlin
mapView.mapboxMap.setCamera(
    CameraOptions.Builder().zoom(14.0).center(point).build()
)
```

点击标注后移动相机：

```kotlin
pointAnnotationManager.addClickListener { annotation ->
    mapView.mapboxMap.setCamera(
        CameraOptions.Builder().zoom(14.0).center(annotation.point).build()
    )
    true
}
```

### Compose：`MapViewportState`

```kotlin
val mapViewportState = rememberMapViewportState {
    setCameraOptions {
        center(Point.fromLngLat(-74.0066, 40.7135))
        zoom(15.5)
    }
}
MapboxMap(mapViewportState = mapViewportState, onMapClickListener = { pt ->
    mapViewportState.easeTo(cameraOptions { center(pt) })
    true
})
```

### 跟随用户位置

需先授权定位，见 [16-地图上显示位置](./16-地图上显示位置.md)。

```kotlin
mapView.location.addOnIndicatorPositionChangedListener {
    mapView.mapboxMap.setCamera(CameraOptions.Builder().center(it).build())
    mapView.gestures.focalPoint = mapView.mapboxMap.pixelForCoordinate(it)
}
```

也可用 **Viewport API** 跟踪位置。

### 适配几何范围

```kotlin
mapboxMap.cameraForCoordinates(
    coordinates = triangleCoordinates,
    camera = cameraOptions { },
    coordinatesPadding = EdgeInsets(50.0, 50.0, 50.0, 50.0),
    maxZoom = null,
    offset = null
) { cameraPosition ->
    mapboxMap.setCamera(cameraPosition)
}
```

## 监听相机变化

```kotlin
val cancelable = mapboxMap.subscribeCameraChanged { event ->
    // 处理变化
}
// cancelable.cancel()
```

Compose：`mapState.cameraChangedEvents.collect { }`

---

## 读取当前相机

```kotlin
val center = mapView.mapboxMap.cameraState.center
```

---

## 限制相机范围

```kotlin
mapboxMap.setBounds(
    CameraBoundsOptions.Builder()
        .bounds(CoordinateBounds(sw, ne, false))
        .minZoom(10.0)
        .build()
)
```

可限制用户只能在某区域（如冰岛、某城市）内平移缩放。
