# 相机动画

> 原文：[Animations](https://docs.mapbox.com/android/maps/guides/camera-and-animation/animations/)

相机动画在一段时间内把 `center`、`bearing`、`pitch`、`zoom`、`padding`、`anchor` 从旧值过渡到新值。

> **注意**：同一时刻，**每种相机属性类型**只能有一个动画在跑；新动画会取消同类型的旧动画。

用 `setCamera()` 会直接跳转，无过渡。

---

## 动画 API 层级

### 高级 API（开箱即用）

通过 `MapboxMap` 或 Compose 的 `MapViewportState` 调用，定制项较少：

| 方法 | 效果 |
|------|------|
| `flyTo` | 缩放+平移，类似飞行 |
| `easeTo` | 平滑过渡 |
| `rotateBy` | 旋转 bearing |
| `scaleBy` | 缩放图上元素视觉大小 |
| `pitchBy` | 改变俯仰 |
| `moveBy` | 快速位移 |

**同时只能有一个高级动画**；新高级动画会取消正在运行的高级动画（即使改的参数不同）。

```kotlin
mapboxMap.easeTo(cameraOptions, mapAnimationOptions)

// Compose
mapViewportState.easeTo(cameraOptions)
```

### 中级 API

创建多个 `CameraAnimator`，用 `playAnimatorsTogether` 或 `playAnimatorsSequentially`：

```kotlin
mapView.camera.apply {
    val bearing = createBearingAnimator(cameraAnimatorOptions(18.0, 20.0) { startValue(15.0) }) {
        duration = 8500
    }
    val pitch = createPitchAnimator(cameraAnimatorOptions(30.0) { startValue(15.0) }) {
        duration = 2000
    }
    playAnimatorsTogether(bearing, pitch)
}
```

Compose 在 `MapEffect` 中使用可能干扰 `MapViewportState`。

### 低级 API

基于 Android `ValueAnimator`，须在 `CameraAnimationsPlugin` 上 **register / unregister**：

```kotlin
val bearing = plugin.createBearingAnimator(cameraAnimatorOptions(160.0) { startValue = 0.0 }) {
    duration = 8500
}
plugin.registerAnimators(bearing)
bearing.start()
// onEnd/onCancel 时 unregisterAnimators
```

## 监听动画事件

### 高级 API

```kotlin
mapboxMap.flyTo(cameraOptions, animatorListener = object : Animator.AnimatorListener {
    override fun onAnimationStart(animation: Animator) {}
    override fun onAnimationEnd(animation: Animator) {}
    override fun onAnimationCancel(animation: Animator) {}
    override fun onAnimationRepeat(animation: Animator) {}
})
```

Compose：`mapViewportState.flyTo(cameraOptions) { isFinished -> }`

### 中低级

`CameraAnimator.addListener(AnimatorListener)`

### 全局生命周期

`mapView.camera.addCameraAnimationsLifecycleListener(...)`

### 单属性变化

- `addCameraCenterChangeListener`
- `addCameraBearingChangeListener`
- `addCameraPitchChangeListener`
- `addCameraZoomChangeListener`
- 等
