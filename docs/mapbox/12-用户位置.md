# 用户位置

> 原文：[User Location](https://docs.mapbox.com/android/maps/guides/user-location/)

Mapbox Maps SDK 可：

- 申请并处理**定位权限**
- 获取设备位置更新
- 在地图上显示**位置圆点（puck）**
- 让相机**跟随**用户移动

## 典型流程

1. `AndroidManifest` 声明 `ACCESS_COARSE_LOCATION` / `ACCESS_FINE_LOCATION`
2. 用 `PermissionsManager` 向用户申请（API 23+）
3. 启用 `location` 组件：`mapView.location.enabled = true`
4. （可选）`viewport.transitionTo(followPuckViewportState)` 跟随用户

## 与 AuraFlow 集成提示

当前项目为 **View + AppCompatActivity**，按 [16-地图上显示位置](./16-地图上显示位置.md) 的 **Android View** 示例集成即可。若后续迁移 Compose，可改用 `MapEffect` + `rememberMapViewportState`。
