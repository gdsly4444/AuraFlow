# 在地图上显示用户位置

> 原文：[User's location on the map](https://docs.mapbox.com/android/maps/guides/user-location/location-on-map/)

显示 puck 前须完成 [14-权限处理](./14-权限处理.md)。

## 启用 Location 组件

### XML

```xml
<com.mapbox.maps.MapView
    app:mapbox_locationComponentEnabled="true"
    app:mapbox_locationComponentPuckBearing="heading" />
```

### 代码（View — 适合 AuraFlow）

```kotlin
mapView.location.apply {
    locationPuck = createDefault2DPuck(withBearing = true)
    enabled = true
    puckBearing = PuckBearing.COURSE
    puckBearingEnabled = true
}
mapView.viewport.transitionTo(
    targetState = mapView.viewport.makeFollowPuckViewportState(),
    transition = mapView.viewport.makeImmediateViewportTransition()
)
```

默认 puck：蓝色圆点 + 白圈，绘制在地图图层内（非普通 Android View 浮层）。

## 自定义 puck

### 脉冲效果

```kotlin
mapView.location.pulsingEnabled = true
```

### 自定义图片（LocationPuck2D）

```kotlin
mapView.location.locationPuck = LocationPuck2D(
    topImage = ImageHolder.from(R.drawable.mapbox_user_icon),
    bearingImage = ImageHolder.from(R.drawable.mapbox_user_bearing_icon),
    shadowImage = ImageHolder.from(R.drawable.mapbox_user_stroke_icon),
    scaleExpression = /* 按 zoom 插值 */
)
```

恢复默认：`createDefault2DPuck()`。

### Bearing 来源

```kotlin
mapView.location.puckBearing = PuckBearing.HEADING  // 设备朝向
mapView.location.puckBearing = PuckBearing.COURSE   // 运动方向
```

## Viewport：相机跟随

`mapView.viewport` 用于跟踪对象（常见为跟随 puck）。

### 内置 ViewportState

| API | 作用 |
|-----|------|
| `makeFollowPuckViewportState(options)` | 相机跟随 puck |
| `makeOverviewViewportState(options)` | 相机框住指定几何 |

### 内置 ViewportTransition

| API | 作用 |
|-----|------|
| `makeDefaultViewportTransition()` | 动画过渡 |
| `makeImmediateViewportTransition()` | 立即跳转 |

### 跟随 puck 示例

```kotlin
val followState = viewportPlugin.makeFollowPuckViewportState(
    FollowPuckViewportStateOptions.Builder()
        .bearing(FollowPuckViewportStateBearing.Constant(0.0))
        .padding(EdgeInsets(topDp, 0.0, 0.0, 0.0))
        .build()
)
viewportPlugin.transitionTo(followState) { success -> }
```

### 总览路线示例

```kotlin
val overviewState = viewportPlugin.makeOverviewViewportState(
    OverviewViewportStateOptions.Builder()
        .geometry(routePoints)
        .padding(EdgeInsets(100.0, 100.0, 100.0, 100.0))
        .build()
)
viewportPlugin.transitionTo(overviewState, viewportPlugin.makeImmediateViewportTransition()) { }
```

## LocationProvider

`LocationComponentPlugin` 通过 `LocationProvider` 接收位置更新。可 `getLocationProvider()` 查看当前实现。

注册 **`LocationConsumer`** 可拿到与 puck 相同的位置流；原始设备定位见 [15-设备定位](./15-设备定位.md)。

### Google Play Services

SDK 可自动使用 **Fused Location Provider**（需 `play-services-location` 依赖）。目标设备无 GMS 时会回退其他实现。

### 自定义 LocationProvider

```kotlin
mapView.location.setLocationProvider(customProvider)
```
